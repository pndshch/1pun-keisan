# 1分けいさん

[Pinda Lab](https://pndshch.github.io/) の子ども向けけいさんドリル。1分間でなんもん解けるかを競うシンプルなブラウザゲーム。

▶ **遊ぶ:** https://pndshch.github.io/1pun-keisan/

## とくちょう

- 4種類のモード: たし算 / ひき算 / かけ算 / ぜんぶ
- 1分タイマー、リアルタイムスコア
- ベストスコア記録（モード別、localStorage）
- X (Twitter) へのスコアシェア
- モバイル / タブレット対応

## 構成

`index.html` 1ファイル完結。依存ゼロ。GitHub Pages で動く。
